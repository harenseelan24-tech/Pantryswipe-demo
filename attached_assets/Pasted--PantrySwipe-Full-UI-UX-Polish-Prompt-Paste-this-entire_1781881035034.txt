## Global Changes (Apply to Every Screen)

### 1. Safe Areas
- Every screen must wrap content in `<SafeAreaView>` with correct edges
- No tappable element should be within 16px of the screen edge
- Bottom tab bar must add `paddingBottom` equal to the safe area bottom inset

### 2. Background & Surface Hierarchy
- Page background: `colors.backgroundLight` / `colors.backgroundDark`
- Cards / elevated surfaces: `colors.surface`
- Modals / sheets: `colors.surfaceElevated`
- Consistent shadow for cards (light mode only):
  ```ts
  shadowColor: '#000', shadowOffset: { width: 0, height: 2 },
  shadowOpacity: 0.08, shadowRadius: 8, elevation: 3
  ```

### 3. No Raw Hex or Magic Numbers
- Remove every hardcoded `#hex` color from components — replace with `colors.*`
- Remove every raw `fontSize`, `padding`, `margin` number — replace with `typography.*` and `spacing.*`

### 4. Icons
- All icons must use `@expo/vector-icons` Feather set at consistent sizes: 20 (inline), 24 (nav/action), 28 (hero)
- No emoji used as structural icons — replace any emoji icons with Feather equivalents
- Icon color must always come from `colors.textPrimary` or a semantic token, never hardcoded

### 5. Touch Targets
- Every tappable element must have a minimum hit area of 44×44pt
- Use `hitSlop={{ top: 8, bottom: 8, left: 8, right: 8 }}` on small icons
- All `Pressable` / `TouchableOpacity` must show pressed feedback: `activeOpacity: 0.72` or a subtle scale (0.97) via `Animated`

## Animation Standards (Apply Globally)

- All state transitions: 200ms, `Easing.out(Easing.cubic)`
- Swipe card throw: 350ms with spring physics (`useSpring` or `Animated.spring`)
- Tab switch: fade crossfade 150ms
- Card entrance (lists): stagger 40ms per item, translate Y from +16px → 0 + fade in
- Loading skeletons: shimmer animation using `Animated.loop` on opacity (0.4 → 0.8 → 0.4), 1200ms duration
- Button press: scale 0.97 on `pressIn`, 1.0 on `pressOut`, 120ms spring
- All `Animated` values must use `useNativeDriver: true` (except layout properties)
- Respect `AccessibilityInfo.isReduceMotionEnabled()` — skip animations when true
