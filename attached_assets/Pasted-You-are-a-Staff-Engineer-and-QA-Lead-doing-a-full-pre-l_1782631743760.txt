PHASE 1 — TYPESCRIPT AUDIT (fix ALL errors first)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Common error categories to look for and fix:

1. Missing 'worklet' directive on Reanimated callbacks:
   ANY function passed to useAnimatedStyle, useDerivedValue,
   useAnimatedScrollHandler, or runOnUI must have 'worklet';
   as its very first line.
   WRONG:
     const style = useAnimatedStyle(() => ({ opacity: val.value }));
   RIGHT:
     const style = useAnimatedStyle(() => {
       'worklet';
       return { opacity: val.value };
     });

2. Feather icon name typos — check these exact names are valid:
   Valid: "arrow-left", "arrow-right", "check", "check-circle",
   "check-square", "square", "heart", "bookmark", "share-2",
   "message-circle", "bell", "settings", "user", "users",
   "compass", "calendar", "clock", "zap", "star", "sun",
   "moon", "coffee", "gift", "target", "droplet", "layers",
   "grid", "scissors", "plus", "plus-circle", "minus", "x",
   "edit-2", "send", "cpu", "image", "map", "map-pin",
   "menu", "more-vertical", "more-horizontal", "search",
   "sliders", "package", "file-text", "chevron-right",
   "chevron-left", "info", "shield", "credit-card",
   "share", "lock", "sunrise", "award", "log-out"
   If a name is not in this list, replace with the closest
   valid Feather icon name.

3. StyleSheet property violations — React Native does not support:
   calc(), vh, vw, rem, em units — use numeric px values
   background-color (camelCase: backgroundColor)
   border-radius (camelCase: borderRadius)
   aspect-ratio shorthand — use { width: X, height: X }
   CSS grid display — use flexDirection and FlatList instead
   textDecoration — use textDecorationLine
   fontVariationSettings — not supported
   boxShadow as a string — use Platform.select shadow props

4. Missing import statements:
   Check every file for used-but-not-imported: Animated,
   useRef, useEffect, useState, Platform, Dimensions,
   StyleSheet, KeyboardAvoidingView, FlatList, TextInput,
   ScrollView, Pressable, TouchableOpacity, SafeAreaView

5. Reanimated 3 API changes:
   OLD (Reanimated 2): useSharedValue, withSpring, etc. from
     'react-native-reanimated' — still valid in v3
   Check: Animated.View from 'react-native' is NOT the same
     as Reanimated's Animated.View. If both are used, alias:
     import Animated as RNAnimated from 'react-native';
     import Animated from 'react-native-reanimated';
   Ensure no file imports Animated from both without aliasing.
