── OPTION 1: SCAN FRIDGE ──
  🤳 "Scan Fridge / Pantry"
  "Point your camera at your food — AI will detect it instantly"
  Background circle: blue (#EBF3FF)

  ON TAP:
    1. Request camera permission via browser getUserMedia API
       Permission prompt uses custom in-app overlay BEFORE triggering browser permission:
         Modal: "📷 Camera Access Needed"
         "PantrySwipe needs your camera to identify food items. We don't store any camera footage."
         Two buttons: "Allow Camera" (blue) | "Not Now" (ghost)
         If "Allow Camera" → triggers browser getUserMedia({ video: { facingMode: 'environment' } })
         If browser denies → show instructions: "Go to Settings → Browser → Camera → Allow"

    2. Camera view opens (full screen):
       Live camera feed with white rounded-rect targeting overlay
       Instructions text: "Point at food items one by one — hold steady for 2 seconds"
       Subtle animated scan line (top to bottom, repeating, blue glow)
       
       Bottom panel (semi-transparent):
         Counter: "Items detected: 3" — updates as AI detects items
         "Done Scanning" button (blue, full width)
         "Switch to Receipt Scan" link

    3. AI Detection (send frame to Claude Vision API every 2 seconds):
       System prompt: "You are a pantry inventory AI. Identify food items visible in this image. For each item provide: name, estimated quantity, unit (pieces/g/ml/pack), and category (dairy/produce/meat/seafood/grains/condiments/sauces/spices/frozen/drinks/snacks/baking). Return only JSON array."
       
       As items are detected, they appear as small animated pills at the bottom of camera view:
         "🥦 Broccoli · 1 head · Produce" (slides up with spring animation)
         "🥛 Milk · 1 carton · Dairy" (slides up)
         "🧀 Cheddar · 200g · Dairy" (slides up)

    4. User taps "Done Scanning" → Review Screen:
       Title: "Here's what we found 👀"
       Subtext: "Edit anything that looks wrong, then add to your pantry."

       List of detected items, each as an editable row:
         Emoji icon + Name (editable text field) + Quantity (editable number) + Unit (dropdown) + Category (dropdown pill)
         Trash icon to remove item from list
         "Looks wrong? Fix it" — tap row to expand and edit all fields

       Bottom: "+ Add another item manually" link

       "Add X Items to Pantry →" button (blue, shows count)

    5. On confirm → items are added to pantry_items table with:
         user_id, name, quantity, unit, category, purchase_date: today, expiry_date: AI estimated or null
         source: 'camera_scan'
       Items appear in Pantry screen immediately, grouped by category
